* Git  
  `git add [文件名]` :保存文件  
  `git commit` :提交更改  
  `git log` :查看提交记录与版本号  
  `git reflog` :查看命令记录  
  `git reset --hard [版本号]` :回退到地版本,高版本的内容被丢弃  
  `git revert -n [版本号]` :以之前的版本为样板创建一个新的版本  
  `git rm [文件名]` :删除文件  
  `git remote add origin [地址]` :将本地仓库添加到远程仓库  
  `git clone [地址]` :将远程仓库克隆到本地  
  `git push` :将本地内容推送到远程仓库  
  `git pull` :将远程仓库的内容同步到本地
